Consider again the diabetes data used in Lab 2 and Application Exercise 6. It is the data used in Efron et al. (2004, http://www.jstor.org/stable/3448465). and provided as part of the lars R package. Let us load these data.

library(lars)
## Loaded lars 1.2
data(diabetes)

The object diabetes is a data frame with 3 variables. We will be working with y as outcome and x as the design matrix: x contains candidate predictors, including age, sex, body mass index (bmi), and mean arterial blood pressure (map) for each of n = 442 diabetes patients, as well as the response of interest, a quantitative measure of disease (diabetes) progression one year after baseline.

Previously, we used the glmnet package to fit the lasso. The lars package provides an alternative, with a focus on the ``trajectory’’ of the coefficients as the regularization constant changes. For instance, the following code estimates lasso regression at all values of the regularization parameter where a predictor is added or removed from the model.

object <- lars(x = diabetes$x, y = diabetes$y)
plot(object)
